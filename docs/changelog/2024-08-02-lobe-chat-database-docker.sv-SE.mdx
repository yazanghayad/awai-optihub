# LobeChat Database Docker Image: Den Sista Pusselbiten för Cloud-distribution

Vi är glada att kunna meddela den officiella lanseringen av den efterlängtade databas Docker-imagen för LobeChat v1.8.0! Detta markerar en betydande milstolpe i våra serverdatabaserbjudanden och ger användarna en komplett cloud-distributionslösning.

## 🚀 Kärnfunktioner

- **Lättviktig Distribution**: Docker-imagen är endast 90MB men erbjuder fullständig databasfunktionalitet.
- **Optimerad Prestanda**: Förkonfigurerad med Server Postgres och NextAuth-autentiseringssystem för att säkerställa optimal anslutningsprestanda.
- **Cloud-synkronisering**: Njut av en sömlös molndatasynkroniseringsupplevelse direkt efter distribution.
- **Flexibel Autentisering**: Stöder integration med tredjepartsleverantörer av SSO-tjänster som Auth0.

## 📘 Uppgraderad Distributionsdokumentation

För att säkerställa att användare kan slutföra distributionen smidigt har vi optimerat strukturen på vår distributionsdokumentation:

- Tydlig introduktion till ramverkskoncept
- Detaljerade distributionsfallstudier
- Omfattande guide för självdistribution

Du kan börja distribuera din egen LobeChat-tjänst genom att besöka den [officiella dokumentationen](https://lobehub.com/en/docs/self-hosting/server-database).

## 🔮 Framtidsutsikter

Vår kunskapsbasfunktion är också under utveckling, så håll utkik efter fler spännande uppdateringar!

Denna uppdatering markerar ett betydande genombrott för LobeChat i cloud-distributionslösningar, vilket gör privat distribution enklare än någonsin. Vi uppskattar communityns tålamod, och vi kommer att fortsätta att sträva efter att ge användarna en bättre upplevelse.
