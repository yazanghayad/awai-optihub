# Stöd för Plugin-system

LobeChat plugin-ekosystemet är en betydande utökning av dess kärnfunktioner, vilket avsevärt förbättrar användbarheten och flexibiliteten hos LobeChat-assistenten.

<Video src="https://github.com/lobehub/lobe-chat/assets/28616219/f29475a3-f346-4196-a435-41a6373ab9e2" />

Genom att utnyttja plugins kan LobeChat-assistenterna få tillgång till och bearbeta realtidsinformation, såsom att söka online efter data och förse användare med aktuella och relevanta insikter.

Dessutom är dessa plugins inte enbart begränsade till nyhetsaggregering; de kan också utökas till andra praktiska funktioner, såsom att snabbt hämta dokument, generera bilder, få data från olika plattformar som Bilibili och Steam, och interagera med en rad tredjepartstjänster.

För att lära dig mer, vänligen se [Plugin-användning](/en/docs/usage/plugins/basic). Dessutom finns högkvalitativa röstalternativ (OpenAI Audio, Microsoft Edge Speech) tillgängliga för att tillgodose användare från olika regioner och kulturella bakgrunder. Användare kan välja lämpliga röster baserat på personliga preferenser eller specifika situationer, vilket ger en personlig kommunikationsupplevelse.
