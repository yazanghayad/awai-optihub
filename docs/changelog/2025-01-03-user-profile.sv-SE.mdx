# Användardatastatistik och Aktivitetsdelning 💯

Vill du veta mer om din aktivitetsprestanda på LobeChat?

Nu kan du få en omfattande förståelse för din AI-data genom statistikfunktionen, och till och med generera personliga aktivitetsdelningsbilder för att dela din LobeChat-aktivitet med vänner.

## 📊 Datastatistik

- **Statistik**: Antal Assistenter / Ämnen / Meddelanden / Totalt Ordantal
- **Rankningar**:
- Modellanvändningsgrad `Top 10`
- Assistentanvändningsgrad `Top 10`
- Ämnesinnehållsvolym `Top 10`
- **Värmekarta**: Aktivitetsdistribution under det senaste året
- **Användaraktivitetsdelning**: Generera personliga aktivitetsdelningsbilder

## 👉 Hur Man Använder

1. Kräver `PgLite` eller `Database`-läge
2. Klicka på din profilbild för att gå in i "Konto" - sidan "Datastatistik"
