# LobeChat 1.0: Ny Arkitektur och Nya Möjligheter

Sedan vi meddelade vårt steg mot version 1.0 i mars har vi varit upptagna med att uppgradera varje aspekt av vår plattform. Efter två månaders intensiv utveckling är vi glada att kunna meddela den officiella lanseringen av LobeChat 1.0! Låt oss ta en titt på våra nya funktioner.

## Stöd för Serverbaserad Databas

Den mest betydande funktionen i LobeChat 1.0 är stödet för serverbaserade databaser. I 0.x-eran gjorde bristen på beständig lagring på serversidan det utmanande, om inte omöjligt, att implementera många funktioner som användare desperat behövde, såsom kunskapsbaser, synkronisering mellan enheter och privata assistentmarknader.

## Hantering av Användarautentisering

I 0.x-eran var den mest efterfrågade funktionen att para ihop med serverbaserade databaser hantering av användarautentisering. Tidigare hade vi integrerat next-auth och Clerk som våra autentiseringslösningar. Som svar på krav på fleranvändarhantering har vi omstrukturerat inställningsgränssnittet till en användarpanel, där vi konsoliderat relevant användarinformation inom det nya användargränssnittet.

## LobeChat Cloud Betatestning

LobeChat Cloud är vår kommersiella version baserad på det öppna LobeChat, och alla funktioner från version 1.0 är nu live i LobeChat Cloud, som har gått in i betatestning. Om du är intresserad kan du gå med i vår väntelista här. Under betatestningsperioden kommer ett begränsat antal åtkomstplatser att släppas dagligen för testningsmöjligheter.
