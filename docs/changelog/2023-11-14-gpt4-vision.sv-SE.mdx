# Stöd för Modeller med Visuell Igenkänning

LobeChat stöder nu flera stora språkmodeller med visuella igenkänningsfunktioner, inklusive OpenAI:s [`gpt-4-vision`](https://platform.openai.com/docs/guides/vision), Google Gemini Pro vision och Zhiyuan GLM-4 Vision. Detta ger LobeChat multimodala interaktionsfunktioner. Användare kan enkelt ladda upp bilder eller dra och släppa dem i chattfönstret, där assistenten kan känna igen bildinnehållet och delta i intelligent dialog, vilket bygger en smartare och mer mångsidig chattupplevelse.

Denna funktion öppnar nya vägar för interaktion, vilket möjliggör kommunikation som sträcker sig bortom text för att inkludera rika visuella element. Oavsett om du delar bilder under vardaglig användning eller tolkar grafik i specifika branscher, levererar assistenten en exceptionell samtalsupplevelse. Dessutom har vi noggrant valt ett urval av högkvalitativa röstalternativ (OpenAI Audio, Microsoft Edge Speech) för att tillgodose användare från olika regioner och kulturella bakgrunder. Användare kan välja en lämplig röst baserat på personliga preferenser eller specifika sammanhang, och därmed få en mer personlig kommunikationsupplevelse.
