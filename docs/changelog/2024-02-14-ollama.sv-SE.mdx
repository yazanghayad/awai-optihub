# Stöd för Ollama-anrop till Lokala Stora Språkmodeller 🦙

Med lanseringen av LobeChat v0.127.0 är vi glada att introducera en fantastisk ny funktion—Ollama AI-stöd! 🤯 Tack vare den robusta infrastrukturen som tillhandahålls av [Ollama AI](https://ollama.ai/) och [communityns insatser](https://github.com/lobehub/lobe-chat/pull/1265), kan du nu interagera med lokala LLMs (Large Language Models) inom LobeChat! 🤩

Vi är glada att avslöja denna revolutionerande funktion för alla LobeChat-användare vid detta speciella tillfälle. Integrationen av Ollama AI representerar inte bara ett betydande tekniskt språng utan bekräftar också vårt engagemang för att kontinuerligt söka mer effektiva och intelligenta sätt att kommunicera med våra användare.

## 💡 Hur Startar Man en Konversation med Lokala LLMs?

Om du står inför utmaningar med privata distributioner rekommenderar vi starkt att prova LobeChat Cloud-tjänsten. Vi erbjuder omfattande modellstöd för att hjälpa dig att enkelt påbörja din AI-konversationsresa.

Upplev den nyuppgraderade LobeChat v1.6 och känn de kraftfulla konversationsfunktionerna som GPT-4 ger!

```bash
docker run -d -p 3210:3210 -e OLLAMA_PROXY_URL=http://host.docker.internal:11434/v1 lobehub/lobe-chat
```

Ja, det är så enkelt! 🤩 Du behöver inte gå igenom komplicerade konfigurationer eller oroa dig för invecklade installationsprocesser. Vi har förberett allt åt dig; bara ett kommando är allt som krävs för att starta djupa konversationer med lokal AI.
