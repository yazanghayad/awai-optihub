# Nya Modellleverantörer Tillagda i LobeChat i November 🎉

Vi är glada att kunna meddela att LobeChat har utökat sitt AI-modellstöd med följande leverantörer:

- **Gitee AI**: [https://ai.gitee.com](https://ai.gitee.com)
- **InternLM**: [https://internlm.intern-ai.org.cn](https://internlm.intern-ai.org.cn)
- **xAI**: [https://x.ai](https://x.ai)
- **Cloudflare Workers AI**: [https://developers.cloudflare.com/workers-ai](https://developers.cloudflare.com/workers-ai)

## Behöver Fler Modellleverantörer?

Skicka gärna in dina förfrågningar på [More Model Provider Support](https://github.com/lobehub/lobe-chat/discussions/6157).
