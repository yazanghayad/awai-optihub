# Stöd för Text-till-bild-generering

Den senaste text-till-bild-genereringsteknologin stöds nu, vilket gör det möjligt för LobeChat-användare att direkt använda text-till-bild-verktyget under konversationer med sin assistent. Genom att utnyttja funktionerna hos AI-verktyg som [`DALL-E 3`](https://openai.com/dall-e-3), [`MidJourney`](https://www.midjourney.com/) och [`Pollinations`](https://pollinations.ai/), kan assistenter nu förvandla dina idéer till bilder. Detta möjliggör en mer intim och uppslukande kreativ process.
