# Större Uppdatering: LobeChat Går In i Artifacts-eran

Vi är glada att kunna meddela den officiella lanseringen av LobeChat v1.19! Denna uppdatering introducerar flera viktiga funktioner som höjer den interaktiva upplevelsen av AI-assistenten.

## 🎨 Artifacts-stöd: Låser Upp Nya Kreativa Dimensioner

I denna version har vi nästan helt replikerat kärnfunktionerna i Claude Artifacts. Nu kan du uppleva följande i LobeChat:

- SVG-grafikgenerering och visning
- HTML-sidgenerering och realtidsrendering
- Dokumentgenerering i fler format

Det är värt att nämna att Python-kodkörningsfunktionen också har utvecklats och kommer att vara tillgänglig i framtida versioner. Vid den tidpunkten kommer användare att kunna använda både Claude Artifacts och OpenAI Code Interpreter, vilket avsevärt förbättrar AI-assistentens praktiska användbarhet.

![Artifacts Funktionsvisning](https://github.com/user-attachments/assets/639ed70b-abc5-476f-9eb0-10c739e5a115)

## 🔍 Ny Upptäcktssida: Utforska Fler Möjligheter

Upptäcktssidan har genomgått en större uppgradering och har nu en rikare variation av innehållskategorier:

- AI-assistentmarknadsplats
- Plugin-showcase
- Modelllista
- Leverantörspresentationer

Denna omdesign ökar inte bara informationstätheten på sidan utan öppnar också ett nytt fönster för användare att utforska AI-funktioner. I framtiden planerar vi att ytterligare utöka funktionaliteten hos upptäcktssidan, potentiellt genom att lägga till:

- Kunskapsbas-delning
- Artifacts-showcases
- Kurerade konversationsdelningar

## 🚀 GitHub Models-stöd: Fler Modellval

Tack vare community-medlemmen [@CloudPassenger](https://github.com/CloudPassenger) för deras bidrag, LobeChat stöder nu GitHub Models-leverantörer. Användare behöver bara:

1. Förbereda en GitHub Personal Access Token (PAT)
2. Konfigurera leverantörsinformation i inställningarna
3. Börja använda gratis modeller tillgängliga på GitHub Models

Tillägget av denna funktion expanderar kraftigt utbudet av modeller som är tillgängliga för användare, vilket ger fler alternativ för AI-konversationer i olika scenarier.

## 🔜 Framtidsutsikter

Vi kommer att fortsätta fokusera på att förbättra funktionaliteten och användarupplevelsen hos LobeChat. I kommande versioner planerar vi att:

- Förbättra Python-kodkörningsfunktionen
- Lägga till stöd för fler typer av Artifacts
- Utöka innehållsdimensionerna hos upptäcktssidan

Tack till varje användare för ert stöd och feedback. Låt oss se fram emot fler överraskningar från LobeChat tillsammans!
