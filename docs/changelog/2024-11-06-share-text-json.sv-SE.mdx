# Uppgraderad Konversationsdelning: Stöd för Textformatexport

I den senaste versionen v1.28.0 har vi lanserat textformatexportfunktionen för konversationsinnehåll, som nu stöder export i både Markdown och OpenAI-format JSON.

Markdown-exportfunktionen möter användarnas behov av att direkt använda konversationsinnehåll i anteckningstagning och dokumentskrivning. Du kan enkelt spara värdefullt konversationsinnehåll och hantera det över olika anteckningsapplikationer för återanvändning.

![Exportera Konversationer som Markdown-text](https://github.com/user-attachments/assets/29508dda-2382-430f-bc81-fb23f02149f8)

Dessutom stöder vi export av konversationer i JSON-format som överensstämmer med OpenAI-meddelandespecifikationer. Detta format kan användas direkt för API-felsökning och fungerar som högkvalitativa träningsdata för modeller.

![Exportera Konversationer som JSON i OpenAI API-specifikation](https://github.com/user-attachments/assets/484f28f4-017c-4ed7-948b-4a8d51f0b63a)

Det är särskilt anmärkningsvärt att vi behåller originaldata från Tools Calling inom konversationen, vilket är avgörande för att förbättra modellens verktygsanropsfunktioner.

Denna uppdatering expanderar kraftigt delnings- och applikationsscenarierna för konversationsinnehåll, och vi hoppas att dessa nya funktioner kommer att förbättra din användarupplevelse.
