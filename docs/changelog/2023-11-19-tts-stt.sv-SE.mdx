# Stöd för TTS & STT Röstkonversationer

LobeChat stöder Text-till-tal (TTS) och Tal-till-text (STT) teknologier, vilket gör det möjligt för vår applikation att omvandla textinformation till tydlig röstutmatning. Användare kan interagera med våra konversationsagenter som om de pratade med en riktig person. Det finns olika röstalternativ för användare att välja mellan, vilket ger rätt ljudkälla för deras assistent. Dessutom, för de som föredrar auditiv inlärning eller söker att få information när de är på språng, erbjuder TTS en utmärkt lösning.

I LobeChat har vi noggrant valt ett urval av högkvalitativa röstalternativ (OpenAI Audio, Microsoft Edge Speech) för att tillgodose användare från olika regioner och kulturella bakgrunder. Användare kan välja lämpliga röster baserat på personliga preferenser eller specifika scenarier, och därmed uppnå en personlig kommunikationsupplevelse.
