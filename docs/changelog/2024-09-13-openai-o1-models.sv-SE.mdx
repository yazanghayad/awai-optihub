# OpenAI O1-seriens Modeller Nu Tillgängliga på LobeChat

Vi är glada att kunna meddela att LobeChat v1.17.0 fullt ut stöder OpenAI:s nyligen lanserade O1-seriens modeller. Oavsett om du är en community edition-användare eller en [Cloud-version](https://lobechat.com) -prenumerant kan du uppleva denna betydande uppdatering.

## Nytt Modellstöd

- ✨ OpenAI o1-preview
- ✨ OpenAI o1-mini

## Förbättrade Funktioner

O1-seriens modeller utmärker sig inom följande områden:

- 💻 Kodskrivning och förståelse
- 🔢 Matematisk problemlösning
- 🎯 Mer precis uppgiftsutförande
- ⚡️ Optimerad prestanda

## Upplev Det Nu

- 🌐 Cloud-version prenumeranter kan börja använda det omedelbart
- 🔧 Självvärdande användare kan börja uppleva det genom att uppdatera till v1.17.0

Denna uppdatering markerar ett viktigt steg för LobeChat i att stödja de senaste AI-modellerna. Vi ser fram emot att se hur O1-seriens modeller kan hjälpa användare att låsa upp nya möjligheter!
